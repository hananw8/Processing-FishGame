# Fish Game (Processing)
A simple 2D survival-style game created in Processing where the player controls a fish and avoids enemies such as sharks.
The player must move, survive, and interact with enemies in real time using keyboard controls.
This project was built as a learning exercise to practice:
- Object-oriented programming
- Game loops
- Player movement and collision handling
- Working with images and sprites
- Managing multiple classes in Processing
# How to Run the Game
1. Download and install Processing from: https://processing.org/
2. Clone or download this repository.
3. Open the `FishGame.pde` file in Processing.
4. Ensure all `.pde` files and image files are in the same project folder.
5. Click the Run button.
# Controls
- `Arrow Keys` - Move player
- `ESC` - Quit the game
